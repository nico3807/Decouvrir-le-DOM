3. ✍️ Modifier le contenu et les attributs

Apprenez à différencier le contenu textuel du HTML et à manipuler les propriétés des balises (src, href, alt).

Exemple Fonctionnel

Ceci est un paragraphe avec HTML.

Lien initial (Site de l'UM)

3.1 Changer le texte ou le HTML

La différence entre textContent et innerHTML est fondamentale pour votre sécurité et votre performance.

Objectif : Remplacer le contenu du paragraphe dans la boîte cible, en ne gardant que le texte. 

// Sélectionnez votre élément au préalable :
const boite = document.querySelector('#target-box');
// Remplacement du contenu par du texte brut (le <strong> disparaît) :
boite.textContent = "Contenu sans formatage HTML.";

Objectif : Remplacer le contenu de la boîte cible en y insérant un nouveau paragraphe avec une balise. 

// Remplacement du contenu par du nouveau HTML :
boite.innerHTML = "<p>Nouveau <strong>contenu HTML</strong> inséré !</p>";

3.2 Changer les attributs (src, href, alt...)

Vous pouvez lire ou modifier les attributs d'une balise HTML (propriétés du DOM) directement via JavaScript.

Objectif : Changer la source (src) et la description accessible (alt) de l'image cible. 

const image = document.querySelector('#image-cible');
// Modifier la source :
image.src = "https://iut-beziers.edu.umontpellier.fr/files/2025/11/affiche-jpo2026-iutbeziers-600x849.png";
// Modifier l'attribut alt :
image.alt = "Nouvelle image de démonstration.";

Objectif : Modifier la destination (href) du lien cible. 

const lien = document.querySelector('#lien-cible');
// Modifier le lien :
lien.href = "https://mmi.fr";
